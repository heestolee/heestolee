  > - 단일 페이지에서도 사용이 가능하고, 여러 페이지의 컨텐츠를 하나의 배포 구성으로 만들수도 있습니다.
  > - 기술 스택 : TypeScript, React 19, Next.js 15, Zustand, TanStack Query v5, Chakra UI, Vercel, Docker, AWS Elastic Beanstalk
